# PELS

PELS (Pris- og Effektstyrt Energilagringssystem) is a Homey Pro app for capacity-aware energy control. It keeps large electrical loads inside your hourly power limit, restores them automatically when headroom returns, and can shift flexible heating toward cheaper hours.

The public documentation lives at [pels.barelysufficient.com](https://pels.barelysufficient.com/).

## Documentation

- User docs and setup guide: <https://pels.barelysufficient.com/>
- Technical reference: <https://pels.barelysufficient.com/technical.html>
